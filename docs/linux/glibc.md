# glibc

```sh
# show version
ldd --version

# On 64-bit Debian based system:
/lib/x86_64-linux-gnu/libc.so.6

# on 32-bit Debian based system:
/lib/i386-linux-gnu/libc.so.6

# On 64-bit Red Hat based system:
/lib64/libc.so.6

# On 32-bit Red Hat based systems:
/lib/libc.so.6
```
